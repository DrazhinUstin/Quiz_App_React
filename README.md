# Quiz App

This project was created using [Create React App](https://github.com/facebook/create-react-app).

## Installation

1. Download zip file or Clone Repo
2. Open your Terminal and `cd` to your local copy of the repo
3. Run `nmp install`
4. Run `npm start`
